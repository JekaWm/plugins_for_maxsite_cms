==========================================
Браузеры и плагины необходимые для работы
=========================================

Полностью функционал подерживают браузеры:
1. Mozilla FireFox 3 (Возможно и более старые весрии)
2. Google Chrome 8 (Возможно и более старые весрии)
3. Safari 4 (не тестировалось)

Без возможности мультивыбора файлов:
1. Internet Explorer
2. Opera

Теоретически должны поддерживаться все современные браузеры с включенными JavaScript, но тесты проводились только на описаных выше.
