Плагин «Редактор категорий» позволяет создавать и редактировать мета-поля у категорий. Также плагин позволяет удобно, путем перетаскивания мышкой, менять у категорий родителей и порядок сортировки.

Функционал частично позаимствован у плагина «category_title».

Реализована возможность перетаскивания страниц между категриями. Плагин дает возможность выделить страницы и массово их удалить или снять с публикации.

На странице редактирования категории есть возможность для настройки отображения категории также, как это делается у страниц - с левым сайдбаром, без сайдбара и т.д.
Для того, чтобы эта функция заработала, необходимо добавить файл main_set_val.php в type/category. (подробнее о такой особенности описано на странице http://max-3000.com/page/maxsite-cms-093)
Добавьте в файл код

$cat_template_mso_set_val = CategoryEditor::getInstance()->template;
if($cat_template_mso_set_val)
{
	if ($fn = mso_fe('main/' . $cat_template_mso_set_val . '/main.php')) 
	{	
		mso_set_val('main_file', $fn); // выставляем путь к файлу
	}
}


При добавлении своего мета-поля ему присваивается ключ. Чтобы получить значения этого поля в любом месте страницы напишите 
CategoryEditor::getInstance()->ВАШ КЛЮЧ

например:
CategoryEditor::getInstance()->template


ВАЖНО: Это самая первая версия плагина. Возможны какие-то недочеты или сбои. О них вы можете сообщать в комментариях. Также возможно могут возникнуть проблемы в совметсной работе с плагином «category_title», поэтому при активации Редактора категорий выключите «category_title». Обмен данными между этими плагинами не предусмотрен. Поэтому при переходе с «category_title» на «Редактор категорий» все тайтлы, дескрипшины и кейворды надо забивать повторно.
