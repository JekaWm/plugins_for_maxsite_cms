Плагин обратного отсчета

Вызывается в прозвольном месте страницы:
	
пример:
	Текст текст текст текст текст текст текст 
	
	[countdown]ПАРАМЕТРЫ[/countdown]
	
	текст текст текст текст текст текст 
	текст текст текст текст текст текст 
	текст текст текст текст текст текст
	
Параметры следующего формата:

	[countdown]ГГГГ/ММ/ДД/Заголовок для счетчика/Текст по окончании отсчета[/countdown]
	
	Дата окончания счетчика:
		ГГГГ - год (2014); Обязательно!
		ММ - месяц (05); Обязательно!
		ДД - день (03); Обязательно!
	
	Дополнительные параметры:
		Заголовок для счетчика - текст перед счетчиком.(заключается в отдельный div#cntdwntitle.);
		Текст по окончании отсчета - текст вместо цифр отсчета, когда текущая дата больше установленной.
		
Важно строго соблюдать параметры.


Разметка:
	
<span class="cndwn-diggit dd-day">ДНЕЙ(цифра)</span><span class="cndwn-string ds-day">ДНЕЙ(текст)</span>
<i>,</i>ПРОБЕЛ
<span class="cndwn-diggit dd-hour">ЧАСОВ(цифра)</span><span class="cndwn-string ds-hour">ЧАСОВ(текст)</span>
<i>,</i>ПРОБЕЛ
<span class="cndwn-diggit dd-min">МИНУТ(цифра)</span><span class="cndwn-string ds-min">МИНУТ(текст)</span>
<i>,</i>ПРОБЕЛ
<span class="cndwn-diggit dd-sec">СЕКУНД(цифра)</span><span class="cndwn-string ds-sec">СЕКУНД(текст)</span>
