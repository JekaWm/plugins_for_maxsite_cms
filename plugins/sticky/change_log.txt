[ История версий для плагина «Sticky» ]
-------------------------------------------------------------------------------

22.01.2014 — 0.1
    — Первая публичная версия.
    — Опции и т.д.

27.02.2014 — 0.2
    — Исправлены мелкие баги в js-файле.
    — Реализован второй («directly» - «прямой») способ залипания виджетов.
    — Добавлена возможность указывать дополнительный css класс для залипшего блока.
    — Добавлен файл с примером стилей для закрипления горизонтального меню в 
      шапке сайте.
    — Доработана страница настройки опций плагина.    

23.03.2014 — 0.3
    — Переработаны алгоритмы залипания и исправлены мелкие баги в js-файле.
    — Реализован третий («alonemenu» - «для меню») способ залипания виджетов.
    — Добавлена возможность указывать callback-функций для залипших блоков.
    — Переработана логика задания опций вызова jquery-плагина.

25.03.2014 — 0.4
    — Исправлены недочёты алгоритмов залипания и исправлены мелкие ошибки в js-файле.
    — Оптимизирована работа в js-файле.
    — Добавлены сопроводительные комментарии в js-файл.

26.06.2014 — 0.5
    — В js-файле удалёно использование $.browser, который более недоступен в jQuery 1.9+

10.01.2015 — 0.6
    — Введены параметры для более тонкой настройки работы с малыми разрешениями экрана
    — Обновлен код для избежания конфликта имён функций с другими плагинами моей разработки

07.07.2015 — 0.7
    — В js-файле исправлен недочёт, приводивший к проблемам залипания при работе в IE
