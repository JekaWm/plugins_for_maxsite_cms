Плагин Кнопка «Наверх»

Если вы хотите использовать вариант плагина, 
не требующий наличия в шаблоне хуков body_start и body_end, 
переместите файл index_no-hook.php из каталога dev в родительский каталог, 
файл index.php наоборот, переместите в каталог dev, после чего переименуйте 
index_no-hook.php в index.php

Вся функциональность сохраняется, за исключением того, что плагин работает 
только при включенном JavaScript.

Также в каталоге dev сохранены файлы изображений в формате PNG,
элементы «конструктора» для создания своих кнопок.
Изображения обработаны в графическом редакторе GIMP.
