Плагин Кросспостинга для MaxSite CMS от Рамира
http://fo.com.ua

Основная функция плагина отправка сообщений на сайт Livejournal.com. В основе идеи лежит плагин Кросспостинга для Вордпресса.

Одна из доработок - это мульти кросспостинг. Т.е. создавая профили в плагине и отмечая разные рубрики можно публиковать статьи с сайта в различные сообщества сообщества.

Со временем в плагин будет добавлена возможность публиковать в другие блоговые системы.

Установка
Скопируйте содержимое архива в папку plugins
Включите плагин в Админ панели - Плагины

Создание профиля
Перейдите на страницу настроек Плагина и заполните поля.

Имя профиля.
Любое понятное название профиля. Например, Сообщество Семинары и тренинги в ЖЖ

Статус по-умолчанию
Включить этот профиль - Отметьте этот пункт, чтобы при создании новой записи, ваш пост был кросс-постирован.
Редактировать кросс-посты - Отметьте этот пункт, чтобы при редактировании записи, ваш пост был изменен.
Удалять кросс-посты - Отметьте этот пункт, чтобы при удалении записи, ваш пост был удален.

Также вы сможете указывать это действие при создании\редактировании записи.

Адрес interface/xmlrpc

Системный адрес для кросс-постирования.

Имя пользователя - имя пользователя в ЖЖ
Пароль - пароль в ЖЖ

Сообщество - сообщество в ЖЖ, в которое опубликовать кросс-пост.
Заголовок - текст, который будет отображаться в верхней части кросс-поста.
Подвал - текст, который будет отображаться в нижней части кросс-поста.
Оставлять комментарии?
Разрешить оставлять комментарии - отметьте, чтобы на ЖЖ можно было комментировать кросс-пост.
Добавлять ссылку для комментирования на этом сайте - вначале кросс-поста добавится надпись из поля "Текст ссылки" со ссылкой на пост на вашем сайте.

Уровень доступа к записям

Публичный - кросс-пост будет виден всем пользователям ЖЖ.

Приватный - будет виден только владельцу аккаунта.

Для друзей - будет виден тем, у кого владелец аккаунта в "Друзьях".

Как обрабатывать cut/xcut?
Ссылка на этот блог - кросс-постировать только аннотацию вашего поста. В ссылке на продолжение указывать текст из поля "Текст ссылки".
Использовать lj-cut - стандартная настройка для ЖЖ. Укажите текст ссылки, ведущей на полную запись в ЖЖ.
Копировать всю запись - запись публикуется в ленте ЖЖ полностью без "ката".

Метки к записи
Использовать рубрики в метках - добавлять названия ваших рубрик к меткам кросс-поста.
Использовать метки в метках
- добавлять ваши метки к кросс-посту.
